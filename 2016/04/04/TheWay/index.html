##TheWay

早上睡了很长很长的一个觉， 做了很多很多梦。 终于， 在浮躁的生命中停下来想了想自己。
想了很重要的几件事：

  • 首先是关于信仰和坚持的。 首先信仰不是一下子突然获得的。一开始， 你没有信仰， 然后你去寻找他， 并在找的路上做出了你你所希望做的事， 在追寻的路上，改变了自己。我觉得这是信仰， 不是最终的结果，事在这个追寻的过程中的收获。
    于是， 我想到了自己，总是去追寻什么， 有时候确实得到了，有时候又没有得到。 然后被这些结果影响着自己的心情。 然而， 过程的美胜过结果着也是我现在对TXY的态度， 我就像安迪（肖申克）一样，每天从墙上挖去一点水泥。 尽管中间可能出现很多意外，没人知道结果会是怎么样的，但是我只是想坚持下去，为了希望， 为了内心的向往，为了走向这个过程中的内心的宁静？
  • 然后是之后自己的发展， 因为前面一个月自己真的没做什么有用的的事， 感觉时间像流水般流过，，所以现在要好好规范一下自己的学习之路，计计划一下自己每天要做的事：
    • ######github的commit
    • ######单词计划
    • ######课程学习
    • ######java／ java web
    • ######数据库
    • ######python学习(python解析)(项目)
